Only applicable to removing stored payment methods.

In this method you should make the network call to tell your server to make a call to the Recurring/<version_number>/disable endpoint. This method is called when the user initiates removing a stored payment method using the remove button.

We provide storedPaymentMethod that contains the id of the stored payment method to be removed in the field StoredPaymentMethod.id.

Asynchronous handling: since this method runs on the main thread, you should make sure the Recurring//disable call and any other long running operation is made on a background thread.

Note that not overriding this method while enabling gift card payments will cause a NotImplementedError to be thrown.

See https://docs.adyen.com/api-explorer/ for more information on the API documentation.
