Drop-in is our pre-built checkout UI for accepting payments. You only need to provide the /paymentMethods response and some configuration data - Drop-in will handle the rest of the payment flow.

To start the payment flow, first extend the DropInService class, and add it to your manifest file. Then call startPayment.
