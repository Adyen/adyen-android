Helper method to transform the activity result into a DropInResult.

The returned value could be:

  • DropInResult.CancelledByUser if the operation was cancelled by the user.

  • DropInResult.Error if an unexpected error has occurred during Drop-in, the DropInResult.Error.reason field will contain the error detail.

  • DropInResult.Error if a DropInServiceResult.Error was returned to the DropInService. The DropInResult.Error.reason field will hold the same value as DropInServiceResult.Error.reason.

  • DropInResult.Finished if a DropInServiceResult.Finished was returned to the DropInService. The DropInResult.Finished.result field will hold the same value as DropInServiceResult.Finished.result.

  • null if the activity result does not correspond to the Drop-in.

Note that DropInResult.Finished will be returned here only if a result intent was provided to the startPayment method.

Return

the result of the Drop-in.
