Configuration class for Installments in Card Component. This class can be used to define installment options for all cards or specific CardType. defaultOptions and cardBasedOptions can be combined together. In that case InstallmentOptions from cardBasedOptions will override the option defined in defaultOptions.

Note: cardBasedOptions should contain only one InstallmentOptions.CardBasedInstallmentOptions instance for a CardType.
