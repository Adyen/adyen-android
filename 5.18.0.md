### Improved
- Redesigned UI for co-badged cards: For BCMC, Carte Bancaire, and Dankort the shopper can select their preferred card brand directly inside the card field, with a description below. For other brands: the card fields shows both brand logos, but the shopper cannot select them.

### Changed
- For the `onSubmit` callback: the size of the payload returned has increased in size, because of changes to the included `sdkData` field.
