interface PaymentComponent<ComponentStateT : PaymentComponentState<out PaymentMethodDetails>?, ConfigurationT : Configuration?> : Component<ComponentStateT, ConfigurationT>

A component that handles collecting user input data. It handles validating and formatting the data for the UI. A valid PaymentComponentState contains PaymentMethodDetails to help compose the payments/ call on the backend.

Should be used attached to a corresponding ComponentView to get data from.
